# Website Beatriz Conradt

Esse website foi desenvolido para a psicologa e psicopedagoga Beatriz Conradt, a qual através de um contrato, concordou com o desenvolvimento deste website.

Tem como objetivo realizar a 3ª avaliação da matéria de desenvolvimento de aplicações I, utilizando as tecnologias aprendidas durante o curso da instituição - UNESC.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
